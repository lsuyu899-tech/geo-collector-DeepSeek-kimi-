## Quick Start (GUI)

1. Open this folder.
2. Double-click `run_app.bat`.
3. Fill your API keys and parameters.
4. Click `开始运行`.

## Quick Start (CLI)

1. Double-click `run_cli.bat`.
2. Follow prompts.
